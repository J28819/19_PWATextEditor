## Local Installation
1. To clone the repository in your computer run this commands.

```bash
git clone git@github.com:J28819/13ORMEcommerceGT.git
//go to the foler
npm i
npm run start:dev
```

2. Or you can access Heroku in this link:  https://pwatexteditor19.herokuapp.com
